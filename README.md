## Front End

- **Kotlin** — Linguagem principal do projeto (usada no front)
- **Jetpack Compose** — Construção de interfaces modernas e reativas
- **Arquitetura MVVM** — Separação entre interface, lógica de negócios e dados
- **Lucas** — Responsável pelo Front End
- **Interface** simples e responsiva
- **Cadastro** de diferentes tipos de serviços
- **Funcionalidades** relacionadas ao agendamento com nome, telefone e e-mail (implementação cliente)
